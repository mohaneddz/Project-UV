AVAILABLE MODELS BY CATEGORY
============================

TIME SERIES
-----------
- SEASONAL_NAIVE (ts.train_seasonal_naive): Seasonal Naive - simple method that repeats the value from the same season in the previous year
- SARIMAX (ts.train_sarimax): Seasonal AutoRegressive Integrated Moving Average with eXogenous factors - statistical model for time series forecasting with seasonality
- PROPHET (ts.train_prophet): Facebook's Prophet - additive model for forecasting time series with trends, seasonality, and holidays

MACHINE LEARNING
----------------
- Linear Regression (ml.train_linear_regression): Linear Regression - assumes linear relationship between features and target
- Random Forest (ml.train_random_forest): Random Forest - ensemble of decision trees with bagging and feature randomness

BOOSTERS
--------
- XGBoost (boosters.train_xgboost) [conditional]: XGBoost - optimized distributed gradient boosting library
- CatBoost (boosters.train_catboost) [conditional]: CatBoost - gradient boosting with categorical feature support and ordered boosting
- AdaBoost (boosters.train_adaboost): AdaBoost - adaptive boosting algorithm that combines weak learners
- Gradient Boosting (boosters.train_gradient_boosting): Gradient Boosting - sequential ensemble method that builds trees to correct previous errors
- LightGBM (ml.train_lightgbm) [conditional]: LightGBM - gradient boosting framework using tree-based learning with histogram-based algorithm

DEEP LEARNING
-------------
- RNN (dl.build_rnn): Recurrent Neural Network - neural network with loops to process sequences
- BI-LSTM (dl.build_bilstm): Bidirectional Long Short-Term Memory - LSTM that processes sequences in both directions
- Transformer (dl.build_transformer): Transformer - attention-based model for sequence processing

TRANSFER LEARNING
-----------------
- TabNet (tl.train_tabnet): TabNet - neural network for tabular data with instance-wise feature selection
- PatchTST (tl.train_patchtst): PatchTST - patch-based time series transformer for long-term forecasting
- TabPFN (tl.train_tabpfn): TabPFN - prior-data fitted network for tabular data

TOTAL MODELS: 16 (15 trained, 1 not trained: TabPFN)

Notes:
- "[conditional]" indicates models that require optional third-party packages.
- This list is intentionally concise and matches the `ALL_MODELS` variable
  defined in `algeria-uv.ipynb`.
- Performance metrics are from the latest training run on Algeria UV data.
